Click HERE if you want to see the technical explanation.


Given the unorthodox use of this Universe Generator's use of 'GET' requests to gather outside HTML for reasons outside of rendering, it is key for CORS to be enabled without triggering browser pre-flights. Many sites (like www.google.com) will deny certain preflight requests altogether made to pages that are meant solely for HTML rending (Google's Search Homepage for example).

Using the browser extension simply enables CORS without triggering pre-flight requests within the browser.
